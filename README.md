# Jagadipa: Petualangan Tri Hita Karana

Game edukasi berbasis web tentang konsep Tri Hita Karana melalui petualangan di halaman sekolah. Pemain memilih lokasi untuk menjalankan misi Parahyangan, Pawongan, dan Palemahan, memainkan minigame, mempelajari materi, dan mengerjakan kuis di akhir. Tersedia buku tumbuhan sebagai materi tambahan setelah menyelesaikan tantangan.

## Uji Coba
Tes game di: `habat-tumbuhan.web.app/`

## Fitur Utama
- Peta interaktif dengan beberapa lokasi misi
- Minigame tematik untuk tiap aspek Tri Hita Karana
- Video/materi pembelajaran terintegrasi
- Kuis evaluasi setelah semua misi selesai
- Buku tumbuhan yang terbuka setelah progres tertentu

## Menjalankan Lokal
- Buka `index.html` langsung di browser, atau
- Jalankan server statis sederhana (misalnya `npx serve`).
